## ğŸ§  Overview

**Probability Simulator** is an interactive Python project for **Class 11 students**.
It allows users to simulate basic probability experiments such as **coin tosses, dice rolls, and card draws**.
It compares **experimental vs theoretical probabilities**, helping students visualize and understand probability concepts.

Key functionalities include:

* Coin toss (fair or biased)
* Dice roll (configurable sides)
* Single or double card draw from a standard 52-card deck
* Comparison of experimental vs theoretical results
* Optional plots for visual understanding

- Simulate **coin toss** (fair or biased)

- Simulate **card draws** (1 or 2 cards)

- Show **theoretical probabilities**
- Optional **plots using matplotlib**
- User-friendly command-line interface
