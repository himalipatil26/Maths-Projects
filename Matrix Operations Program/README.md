Matrix Operations Program is an interactive Python project for Class 11 students.
It allows users to perform basic matrix operations such as addition, subtraction, multiplication, transpose, determinant calculation, and solving linear equations.

Key functionalities include:
- Matrix addition, subtraction, and multiplication
- Transpose of any matrix
- Determinant of a square matrix
- Optional: solve linear equations (Ax = b)
- User-friendly command-line interface
